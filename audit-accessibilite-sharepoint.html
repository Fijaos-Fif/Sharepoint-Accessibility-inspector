👋 Bienvenue !

Activons l'outil en 2 minutes.

Étape 1 — Installer Python (si pas déjà fait)

Python est gratuit. Sur un poste d'entreprise, passez par votre centre logiciel ; sinon python.org.

  1. Ouvrez le centre logiciel de votre organisation (ou python.org)
  2. Cherchez « Python »
  3. Cliquez « Installer » — ~2 minutes

Étape 2 — Démarrer l'outil

Dans le dossier de l'outil, double-cliquez sur :

  • Mac : Démarrer (Mac).command
    Si Mac demande une confirmation : clic-droit → Ouvrir → Confirmer
  • Windows : Démarrer (Windows).bat

Une fenêtre s'ouvre, laissez-la tourner. L'outil démarrera automatiquement dans votre navigateur.

Étape 3 — Vérifier

Une fois l'outil démarré, cliquez sur le bouton ci-dessous. Il vérifiera que tout est en place et vous redirigera automatiquement.

Étape 4 (optionnel) — Connecter votre service d'IA

Pour activer les suggestions IA (textes alternatifs, reformulations…), renseignez l'URL « chat completions » de votre portail IA interne, votre clé API et le modèle.

Vous pouvez retrouver toutes ces infos dans l'onglet 📖 Aide à tout moment.

📌 Ajouter « Auditer cette page » à vos favoris

Une fois ajouté, ce favori capturera n'importe quelle page SharePoint en un clic. Plus besoin de copier-coller le code à chaque fois.

Méthode 1 — Glisser-déposer (la plus rapide)

Glissez le bouton bleu ci-dessous directement dans votre barre de favoris :

📤 Auditer la page

⚠️ Sur Arc, le glisser-déposer ne marche pas — utilisez la méthode 2.

Méthode 2 — Copier-coller (Arc, ou si la méthode 1 ne marche pas)

  1. Cliquez le bouton « Copier le code » ci-dessous
  2. Créez un nouveau favori dans votre navigateur :
    • Chrome / Edge / Safari : appuyez Cmd+D (Mac) ou Ctrl+D (Windows)
    • Arc : menu Arc en haut à gauche → Bookmarks → Add Bookmark
  3. Donnez-lui le nom « Auditer la page »
  4. Dans le champ « Adresse » ou « URL », supprimez le contenu actuel et collez le code (Cmd+V)
  5. Sauvegardez. C'est fait !
